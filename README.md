Project description:This project is a console-based Java application that simulates basic zoo management.
It demonstrates core Object-Oriented Programming principles using animals, a zoo, and a zookeeper.

The project includes: continue working with my subject area;
follow data abstraction principle;
organizing a data pool, applying filtering, searching and sorting;
implement encapsulation, inheritance and polymorphism;
override toString (), equals (), hashCode () methods.
